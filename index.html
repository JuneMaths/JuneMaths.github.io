Recent Work

➣ Python programs

1. Exchange rate converter

The program helps to calculate currency exchange. This displays the gui window where you can calculate the final amount after entering all the values. It retrieves current data from the website of the Polish National Bank (NBP).

2. Winnie th Pooh Game

The goal of the game is to help the always hungry Winnie the Pooh get more and more honey. However, you have to watch out for the bees and avoid them during the flight. You score points for each honey you caught (The number of points earn: the amount of honey collected * the number of lives left) and lives are deducted for each hit bee. You only have three lives. If you run out of lives in the game and you managed to beat the record, your score is saved in the file with the highest scores.

3. Position in the text

The program finds a position of any sign in a sentence. It returns a number of the sign in the sentence and positions of the sign. It plots histogram of numbers of every signs in the sentence.

4. Oblique projectile motion without air resistance

The program calculate a trajectory of oblique projectile motion without air resistance. The model takes into account change of the gravity depending on the change in height. It plots the result on the graph and returns the falling time, maximum height and the throw range.

5. Disks board

The program draws 100 random disks of the same radius on a 15x15 board. It checks if there is any collision between two disks. If the answer is true. It moves a disk, changes it's color and checks again.

➣ MATLAB programs

1. Tangent to the circle

The program draws a tangent to a circle with any given initial parameters (circle center O(x0,y0) and radius R) from any initial point P(x1,y1)

2. Tangent plane to a three-dimensional function

The program draws tangent plane to a function of two variables passed as an anonymous function. It is the first argument of the function. The second argument is a metod of drawing the function. It can be drawn in a circle or a rectangle. The third argument is a list of necessary data for that method. The last argument is a list of point values from the function for which we want to draw a tangent plane.
